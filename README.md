# Projet HDR25 - Power BI

## Description
Ce projet a pour objectif de créer un **dashboard interactif Power BI** sur l'Indice de Développement Humain (HDI) et ses composantes.  
Le dashboard analysera :  
- Le développement humain global par pays et par catégorie (Very High, High, Medium, Low)  
- Les tendances de l'IDH entre 1990 et 2023  
- Les inégalités ajustées (IHDI)  
- Les indicateurs de genre (GDI, GII)  
- L'impact des pressions planétaires sur le développement humain  


## Sources de données
- Human Development Index (HDI) 2023 – PNUD  
- Inequality-adjusted Human Development Index (IHDI)  
- Gender Development Index (GDI)  
- Gender Inequality Index (GII)  
- Planetary pressures-adjusted HDI  

## Objectifs du dashboard
- Visualiser les indicateurs HDI par pays et catégorie  
- Comparer les tendances historiques (1990-2023)  
- Identifier les inégalités et les écarts entre revenu et développement humain  
- Mettre en évidence les indicateurs de santé, éducation et revenu
